File Education

Common Android file formats explained

Knowing file extensions helps you avoid opening the wrong file, deleting important backups, or trying to play media in an app that does not support it.

Quick reference

FormatUsed forRecommended action
APKAndroid app installer.Verify source before installing.
ZIPCompressed files or backups.Extract only trusted archives.
MP4 / MKVVideo files.Open in a media player.
MP3 / M4AAudio files.Open in a music or playlist player.
PDF / TXTDocuments and notes.Keep in a Documents folder.

Organization tip

Sort files by type before cleanup. APKs belong in an installer folder, videos in Movies, music in Music, and documents in Documents. This makes TV remote navigation much easier.
