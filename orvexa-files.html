Use cases, permissions, and safety

Common use cases

  • Find downloaded APKs on Android TV or Fire TV.
  • Organize videos, music, documents, and transfer folders.
  • Review files after using a sharing app or USB drive.

Permissions explained

Storage and media permissions are used to show local files and folders. APK handling requires install-package access only when you choose to install a file. Orvexa Files is designed around local device management and does not require an account.

Quick fixes

  • If folders appear empty, check Android storage permission in system settings.
  • If an APK will not install, confirm unknown-app access and Android version compatibility.
  • If storage is full, remove old downloads before copying new files.
Does Orvexa Files upload my files?

No. It is designed for local browsing and file management on your device.

Can I use it with USB storage?

Yes, when the Android device exposes the USB drive through its storage system and permissions are granted.

Why can I see some folders but not others?

Android versions handle storage permissions differently. Check app permissions, try the system file picker when available, and review the file permissions guide.

Should I delete APK files after installing?

Usually yes. Keep only versioned backups you intentionally need, then remove old installers to save Android TV storage.

What should I do before clearing a folder?

Open a few files, check dates and sizes, and avoid deleting app data or unknown folders until you identify what created them.
