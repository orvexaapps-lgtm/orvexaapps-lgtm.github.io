Permissions

How Android file permissions work

Modern Android separates storage access by purpose. A file manager needs permission to show files, a media player needs permission to read media, and an installer needs permission only when installing an APK.

Common permission types

Photos and videos

Allows an app to find and open image or video files.

Music and audio

Allows local players to list and play audio files.

Files and folders

Used by file managers to browse documents, APKs, archives, and storage directories.

Install unknown apps

Lets one selected app request APK installation. Keep it disabled when not needed.

Permission safety checklist

  1. Grant permissions only when the app feature requires them.
  2. Review permissions again after app updates.
  3. Remove unknown-app installation access after sideloading.
  4. Use app pages and privacy policies to understand why access is requested.
