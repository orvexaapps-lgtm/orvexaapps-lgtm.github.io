Use cases, privacy, and troubleshooting

Use cases

  • Listen to public radio stations on Android TV, Fire TV, phones, and tablets.
  • Save favorite stations for faster access during daily listening.
  • Use a TV-friendly layout for living-room audio playback.

Permissions explained

Internet access is required to stream radio stations. Background playback services allow listening while using supported device features. Advertising identifiers may be used where ads or analytics are included.

Changelog

  • Version 1.0.2: improved station browsing and playback reliability.
  • Version 1.0.0: first public release with search, favorites, and radio playback.

Troubleshooting

  • If a station fails, try another station because public streams can go offline.
  • If playback stops, check Wi-Fi strength and battery or background restrictions.
  • If search returns little, try country, language, or genre names.
Does Orvexa Radio host the stations?

No. The app helps users access public radio streams, and availability may change when stations update their streams.

Can I use it without internet?

No. Live radio streaming requires a network connection.

Why does one station work and another fail?

Public radio streams can change, go offline, block regions, or use formats that a device handles differently.

Does saving a favorite download the station?

No. Favorites make stations easier to find again; they do not store live radio audio for offline use.

What helps radio playback stability?

Use a strong Wi-Fi connection, avoid running too many background apps, and restart the app when a stream becomes stale.
