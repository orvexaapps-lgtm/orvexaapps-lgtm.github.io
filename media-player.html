Permissions explained

Troubleshooting

  • If a file will not play, confirm the codec is supported by the device.
  • If media is missing, rescan storage or move files into a standard Movies or Music folder.
  • If playback stutters, copy the file to internal storage or use a faster USB drive.
Does Orvexa Media Player stream online videos?

It is primarily focused on local playback from files stored on the device or accessible storage.

Why does a video play on my phone but not on TV?

Android TV devices vary in codec support. Re-encode the file or try a format the TV supports natively.

Why does playback buffer for a local file?

Local stutter can happen when USB storage is slow, the file bitrate is high, or the device struggles with the codec.

How should I organize media files?

Keep videos in Movies, songs in Music, and temporary downloads separate so the player and file manager can find content predictably.

Can I use it for offline playback?

Yes. Files already stored on the device or accessible storage can be played without streaming from the internet.
